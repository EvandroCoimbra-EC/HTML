desenvolvimento web

O que e desenvolvimento web ?

O desenvolvimento web inclui criação, codificação com sistemas e programação de sites e seus respectivos elementos , e linguagem. Ele tambem envolve integração com sistemas e ferramentas, armazenamento de bancos de dados, cibersegurança, entre outros, com o objetivo de estruturar cada detalhe de uma página da internet.

O que faz um desenvolvedor web ?

O desenvolvedor de sites, chamado de profissional de Dev ou web Developer, é o responsavel por criar todo o planejamento estrutural de paginas, utilizando linguagens de programação como HTML, CSS, JavaScript, PHP, entre varias outras.
